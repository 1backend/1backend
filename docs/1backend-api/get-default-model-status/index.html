Get Default Model Status

GET 
/model-svc/default-model/status

Retrieves the status of the default model.

Requires the model-svc:model:view permission.

Responses​

  • 200
  • 401
  • 500

Model status retrieved successfully

Unauthorized

Internal Server Error
